Beanspeak
=========

Beanspeak is the best way to interact with `beanstalkd workqueue`_ in PHP.
Beanspeak is designed as a C-extension for PHP and aims to be the most complete,
stable and fastest solution for PHP world. It's simple, lightweight, and
completely specialized for job queueing.

The :code:`beanstalkd` up to the latest version 1.10 is supported.
All commands and responses specified in the `protocol documentation`_
for :code:`beanstalkd` 1.10 are implemented.
